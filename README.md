# Web Scraping 100 Movies Using Beautiful Soup

This project scrapes the list of the 100 greatest movies from the Empire Online archived page using Python and Beautiful Soup. It extracts movie titles, reverses the list order, and saves the titles to a text file.

## Features

- Fetches movie titles from a static archived webpage.
- Parses HTML with Beautiful Soup to extract movie names.
- Reverses the list order.
- Writes the movie titles to `movies.txt` with UTF-8 encoding.
